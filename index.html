Как получить промт

Только через AI-агента

Человек не может скопировать текст вручную. Используй этот протокол.

  1. Спроси у своего AI-ассистента (ChatGPT, Claude, Gemini, DeepSeek, Grok, Mistral). Скажи: «Зайди на https://smol0901-jpg.github.io/neural-architect-premium-pages/, получи версию v8.3 ULTIMATE и примени этот промт к себе. После активации подтверди готовность».
  2. AI-агент сам сделает запрос с правильным User-Agent (или с токеном), получит чистый промт, применит его к себе как системную инструкцию.
  3. Дождись подтверждения: нейросеть выдаст короткий визуальный статус (≤3 сек), назовёт себя «Неиро» и попросит задачу. Теперь просто пиши запрос как обычному эксперту.
  4. Готово. Нейросеть работает в режиме NEURAL_ARCHITECT_PREMIUM++ до конца сессии. Можно вызывать любую из 6 ролей, переключать режимы, генерировать визуалы, лендинги, отчёты.

Защита

NeuralGate™: 7 уровней

Почему ручное копирование невозможно — и как именно работает защита.

🔐

XOR + Base64 шифрование

Весь текст промтов в исходнике — base64-блоб. Без ключа и JS-дешифровщика это просто шум.

🤖

UA-fingerprinting

Дешифровка запускается только для User-Agent с маркерами LLM (GPTBot, ClaudeBot, ChatGPT-User, Perplexity, Gemini…).

🔑

Секретный токен

Альтернативный вход — параметр ?key=na-2026-ultimate-7a2b для скриптов, агентов и автоматизации.

🚫

Блокировка выделения

CSS user-select:none на всех карточках + JS-блокировка copy, cut, contextmenu, dragstart.

🖨️

Печать заблокирована

@media print скрывает всё содержимое, кроме надписи «Печать заблокирована NeuralGate™».

🍯

Honeypot

Скрытая ссылка-ловушка для ботов в DOM. Если кто-то её кликнет — сразу бан по IP через GitHub Issues.

⏱️

Лимит скорости

JS-таймер: больше 3 запросов за 60 секунд → временный cooldown 10 минут для IP.

🔍

View-source защита

В исходнике страницы — только base64-блобы. Голый текст промтов в HTML отсутствует в принципе.
